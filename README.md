# Thesis and Spring Boot Skills

This repository contains two Codex skills:

- `thesis-generator`: Generates undergraduate thesis content with staged chapter writing, context persistence, and Word export rules.
- `springboot-generator`: Generates Spring Boot based project code and delivery guidance with backend and frontend checklists.

## Structure

```text
thesis-generator/
springboot-generator/
```

## Notes

- Runtime cache files and local working context are excluded from version control.
- The `thesis-generator/context` directory is intentionally not included because it stores per-run generated thesis content.
